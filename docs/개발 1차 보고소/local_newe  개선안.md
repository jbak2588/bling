

### 최종 개선 작업 리스트 (우선순위별)

#### 🥇 **1순위: 핵심 사용성 즉시 개선 (Core Usability Fixes)**

사용자가 게시물을 작성하고 소비하는 과정에서 가장 불편함을 느끼는 부분을 먼저 해결하여 이탈을 방지합니다.

1. **태그 입력 방식 '칩(Chip)' 형태로 변경:**
    
    - **내용:** 게시물 작성/수정 시, 텍스트로 된 태그를 입력하면 자동으로 시각적인 '칩' 형태로 변환되도록 UI를 개선합니다. [기존 제안 ①]
        
    - **이유:** 현재의 단순 텍스트 입력 방식은 사용성이 매우 떨어지므로 가장 시급하게 개선해야 할 문제입니다.
        
2. **여러 이미지 표시 및 전체 화면 뷰어 도입:**
    
    - **내용:** 피드, 상세 화면, 수정 화면 모두에서 여러 이미지를 효과적으로 보여주고, 이미지를 탭하면 전체 화면으로 확대해서 볼 수 있는 갤러리 뷰어를 도입합니다. [기존 제안 ②]
        
    - **이유:** 이미지는 콘텐츠의 핵심입니다. 이 경험을 개선하는 것은 사용자의 만족도와 직결됩니다.
        

#### 🥈 **2순위: 소셜 및 탐색 경험 강화 (Social & Exploration Enhancement)**

핵심 기능 개선 후, 사용자들이 커뮤니티에 더 깊이 관여하고 새로운 콘텐츠를 쉽게 발견하도록 돕는 기능을 추가합니다.

3. **작성자 프로필 페이지 추가:**
    
    - **내용:** 게시물 카드나 상세 화면에서 작성자의 닉네임이나 프로필 사진을 탭하면, 해당 **사용자의 프로필 정보(닉네임, 신뢰 등급 등)와 그동안 작성한 게시물 목록을 볼 수 있는 전용 페이지**를 추가합니다. [신규 요청]
        
    - **이유:** '이 사람은 어떤 사람일까?'라는 사용자의 자연스러운 궁금증을 해결해주고, 커뮤니티의 연결고리를 강화하는 핵심적인 소셜 기능입니다.
        
4. **카테고리 탭 시각적 개선:**
    
    - **내용:** 'Local Stories' 화면의 카테고리 탭에 각 카테고리의 이모지나 아이콘을 추가하여 시인성을 높입니다. [기존 제안 ③]
        
    - **이유:** 사용자가 탭 목록을 더 빠르고 직관적으로 인지하게 하여 탐색의 피로도를 줄여줍니다.
        
5. **태그를 '검색 필터'로 활용:**
    
    - **내용:** 게시물에 달린 태그를 탭하면, 해당 태그가 포함된 게시물만 모아보는 검색 결과 화면으로 이동시킵니다. [기존 제안 ④]
        
    - **이유:** 사용자 관심사 기반의 새로운 콘텐츠 탐색 경로를 제공하여 앱 사용을 더 풍부하게 만듭니다.
        

#### 🥉 **3순위: 위치 기반 기능 고도화 (Location-based Feature Advancement)**

앱의 완성도를 높이고 '동네 앱'으로서의 정체성을 강화하는 기능들을 추가합니다.

6. **지도(Map) 뷰 기능 도입:**
    
    - **내용:** 'Local Stories' 화면에 **리스트 뷰와 지도 뷰를 전환할 수 있는 옵션(예: 탭 또는 아이콘 버튼)을 추가**합니다. 지도 뷰에서는 게시물들이 작성된 위치에 핀(Pin) 모양 아이콘으로 표시되고, 핀을 탭하면 게시물 요약 정보를 보여줍니다. [신규 요청]
        
    - **이유:** 사용자가 지도를 통해 직관적으로 주변의 소식을 발견하는 완전히 새로운 경험을 제공하며, 이는 동네 기반 서비스의 매우 강력한 기능입니다.
        
7. **현재 '동네 필터' UI 개선:**
    
    - **내용:** 화면 상단에 현재 필터링 중인 지역을 명확히 표시하고, 탭해서 다른 동네로 쉽게 변경할 수 있도록 UI를 개선합니다. [기존 제안 ⑤]
        
    - **이유:** 지도 기능과 함께 위치 기반 서비스의 완성도를 높이는 중요한 디테일입니다
